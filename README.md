# Open

Deep link redirector via GitHub Pages.

## Usage

### Generic (query param)
```
https://keenranger.github.io/open/?url=kakaomap://route?sp=37.5,127.0&ep=37.4,127.1
```

### Presets (short URLs)
```
https://keenranger.github.io/open/home     → Home
https://keenranger.github.io/open/work     → Work
```

## How it works

Static HTML + JavaScript redirect. No server needed.
